Features
What Startup Auditor Does
Registry Scan
Reads all Windows Run and RunOnce keys — HKCU and HKLM — to find every program set to launch at login via the registry.
Startup Folder Scan
Checks both the user and all-users Startup folders for shortcuts that auto-launch when Windows loads.
Scheduled Task Scan
Queries all scheduled tasks with logon triggers — the startup entries most tools completely miss.
Risk Scoring
Every entry gets a Low, Medium, or High risk rating based on publisher trust, file location, digital signature, and name patterns.
Plain-English Explanations
Click any row to read a plain-language description of what that entry does and whether you should be concerned — no tech jargon.
Publisher Verification
Resolves the publisher name for every executable and flags entries with no verified publisher or digital signature.
Disable & Enable
Disable any startup entry with one click. The entry is backed up and can be re-enabled at any time — nothing is deleted.
Quarantine & Restore
Fully remove suspicious entries from startup and store them safely. Stronger than Disable — always restorable with one click.
HTML Audit Report
Export a full styled HTML report with risk summary cards, all entries, file details, and plain-English explanations.
CSV Export
Export all entries to a CSV spreadsheet including risk scores, publisher info, file size, MD5 hash, and signature status.
Risk Ratings
How Risk Scoring Works
● Low Risk
Known publisher, valid digital signature, installed in a standard location. Safe to leave running.
● Medium Risk
One suspicious flag — such as an unknown publisher or missing signature. Review before deciding.
● High Risk
Runs from a temp folder, has a randomly generated name, or two or more suspicious flags. Quarantine recommended.
